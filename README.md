# Team Generator - Gaming Edition

Generate balanced teams for your gaming sessions with our futuristic gaming-themed team generator.

## Features
- **Player Management**: Easily add, edit, and remove players with custom names.
- **Skill Levels**: Assign skill levels (Noob, Casual, Pro, God Mode) to players.
- **Role Selection**: Select gaming roles (Tank, DPS, Support, Sniper, Flex).
- **Status Toggle**: Toggle player availability (Active/Inactive) for the generator pool.
- **Futuristic Gaming UI**: Premium glassmorphic interface with immersive animations and sound effects.
- **Balanced Team Generation**: Automatically generates balanced teams based on player skill levels.

## Tech Stack
- **Vite** (Build tool)
- **TypeScript** (Language)
- **React** (UI library)
- **shadcn-ui** / **Radix UI** (Components)
- **Tailwind CSS** (Styling)

## Local Development

Ensure you have [Node.js](https://nodejs.org/) installed.

1. Install dependencies:
   ```sh
   npm install
   ```

2. Start the local development server:
   ```sh
   npm run dev
   ```

3. Open [http://localhost:8080](http://localhost:8080) in your browser.

## Project Structure
* `src/components`: UI components.
* `src/pages`: Page views.
* `src/hooks`: Custom React hooks.
* `src/lib`: Utilities and helpers.
