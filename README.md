# Team Generator

The app lives in [`teamgen/`](teamgen/) — **TeamGen**, a modern web app that splits any group into fair random teams. Quick name entry or CSV/Excel import, flexible splitting, late additions, optional score keeping, and Google sign-in to save sessions.

```sh
cd teamgen
npm install
npm run dev
```

See [`teamgen/README.md`](teamgen/README.md) for features and Firebase setup.
