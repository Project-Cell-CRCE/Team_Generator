# 🧩 TeamGen

**Split any group into fair random teams in seconds.**
Type or paste names — or import a CSV/Excel sheet for big lists — pick how to split, and shuffle. Latecomers slot into the smallest team automatically, and you can keep score while you play.

### ✨ What it does

- 📝 **Quick entry** — type a name and press Enter, or paste a whole list
- 📊 **CSV / Excel import** — drag-and-drop `.csv`, `.xlsx`, or `.xls`, with a name-column picker
- ⚖️ **Flexible splitting** — by number of teams or team size, with a live preview
- ⏰ **Late additions** — new arrivals join the smallest team automatically
- 🏆 **Optional scoring** — points per team with a live leaderboard
- 📤 **Export anywhere** — copy as text, or download CSV, styled Excel, or PDF
- 🔐 **Optional Google sign-in** — sessions auto-save to the cloud; guests can use everything without an account
- 🌓 **Modern, responsive UI** — light/dark themes, works great on a phone at the sideline

### 🚀 Quick start

```sh
cd teamgen
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — the app runs fully in guest mode with zero configuration.

📖 See [`teamgen/README.md`](teamgen/README.md) for full feature details and Firebase setup (needed only to enable saved sessions).
